IMD Confiuration Script

## Overview
This unofficial script interacts with the the Vertiv™ Geist™ IMD ([Interchangeable Monitoring Device](https://www.vertiv.com/globalassets/products/critical-power/power-distribution/vertiv-geist-rpdu-imd-data-sheet-en.pdf)) API for configuration and firmware upgrades. It is compatible with IMD-03x firmware versions 5 and 6.
For API reference, see the [Geist™ API Specification](https://www.vertiv.com/4a5013/globalassets/products/critical-power/power-distribution/geist-api-specification-api-specifications-sl-70874.pdf).
